제주대학교 VR&AR Challenge <1등>

제주대학교에서 진행한 VR&AR Challenge로 전공자와 비전공자들이 협업하여 제주도를 주제로한 컨텐츠 개발 Challenge
본인이 속한 팀은 제주도 축제를 주제로 관광 컨텐츠를 개발하였습니다.
cospaces라는 VR&AR 교육용 사이트에서 작업을 진행하였고 개발은 블록코딩과 JavaScript를 사용하였습니다.
팀에서 전반적인 기획, 축제 1 공간 구현, 전체 코드 개발 담당, 간단한 3d object 제작을 수행하였습니다.

2021 창의적 종합설계 경진대회 <장려상>

'4차산업혁명과스마트관광' 수업에서 진행한 프로젝트로 Unity 엔진을 이용하여 AR 슈팅 게임을 개발했습니다.
스마트폰의 카메라를 통하여 현실에 배경에 등장하는 적을 제거하여 점수를 얻는 방식의 게임입니다.
주어진 적을 모두 제거하면 다음 라운드로 이동하고 라운드가 넘어갈수록 제거해야 하는 적의 수가 늘어나는 방식입니다.
해당 교과목에서 작업물에 대하여 1등을 차지하여 상을 받게 되었습니다. (장려상인 이유는 해당 경진대회에서 교과목에 부여할 수 있는 최대 상이었기 때문입니다.)

2022 컴퓨터공학전공 학술 전시회 <장려상>

학술동아리 <분과> 부원들과 함께 준비한 프로젝트로 그 중 소속팀인 앱 개발팀에 소속한 5명이 준비한 전시회 출품작 [응애! 저 내려요!]
해당 앱의 경우 버스를 이용하는 자녀가 어디서 탑승하고 내렸는지 보호자에게 알려주고
보호자가 자녀에게 출발 정거장과 도착 정거장에 대한 정보를 앱에 입력하면 자녀가 해당 정보를 확인하고
버스 탑승을 하는데 문제가 발생하지 않게 도와주는 앱입니다.
frontend의 경우 react-native를 사용, backend의 경우 firebase를 이용하여 기능을 구현하였습니다.
앱의 UI디자인, 로그인/회원가입 기능 구현, 보호자/자녀 역할 선택, 메인 화면에서의 여러 기능 개발을 담당하였습니다.

2022 제1회 AI 자율주행 드론 경진대회 <대상>

자율주행 AI 드론 교육을 듣고난 뒤 참여한 대회로 Tello 드론을 이용하여 여러 문제를 해결하는 방식으로 진행되었습니다.
opencv를 이용하여 드론에 장착되어있는 카메라를 통해 특정 물체와 화살표 등을 이용하여 드론의 진행 방향 제어를 작업하였습니다.
Teachable Machine을 이용하여 카메라를 통해 받는 정보를 머신러닝 모델로 만들어 적용시키는 작업도 진행하였습니다.

아노따르 : ANOTAR(온라인 전시회)

ANOTAR<학술 활동의 작품화: 메타버스로의 안내> 라는 이름으로 전국 여러 대학교에 재학 중인 학생들이 모여 메타버스라는 주제를 가지고 온라인 전시회를 준비하였습니다.
총 14명의 대학생이 자신의 분야를 선택하여 각자 기획, 학술, 개발, 디자인, 홍보 부서로 나뉘어 작업을 진행하였습니다.
Cospaces라는 VR&AR 교육용 사이트에서 작업을 진행하였고 기본 개발 설계는 블럭 코딩과 JavaScript로 이루어졌습니다.
개발 부서에서 4개의 담당 구역을 개발하였고 다른 개발부원들의 작업물에 대하여 피드백하고 전반적으로 수정하는 역할을 수행하였습니다.
전시회 시작 이후 오류 수정 등의 관리 작업도 진행하였습니다.
