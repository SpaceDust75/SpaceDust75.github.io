Ongoing Project

현재 진행중인 프로젝트 목록

2023년 프로젝트 진행 목록

전공 과목 프로젝트 & 개발 팀 프로젝트 작업

Unreal Engine 5를 이용한 게임 개발

개발 인원 5명으로 이루어진 Unreal Engine 5를 이용한 게임 개발 팀 '녹은리얼'의 프로젝트 입니다.
모든 인원이 기획, 개발 등 전반적인 작업에 참여하는 프로젝트입니다.
현재 기획은 마무리 단계이고 다가오는 5월부터 개발을 진행합니다.
3D 로그라이크 TPS 장르의 게임을 준비하고 있으며 단기간 목표는 포트폴리오용 게임 개발이며
장기 목표는 개발한 게임을 좀 더 개선하여 출시하는 것이 목표인 프로젝트입니다.

Arduino를 이용한 외골격 제품 개발

6명의 팀원(기획, 문서 관리, 개발, 디자인 부서)으로 이루어진 제품 개발 팀입니다.
일상에서 편하게 사용할 수 있는 외골격 로봇을 만드는 것이 목표인 프로젝트로써
강화학습을 이용하여 사용자의 움직임 정보를 받아 사용자가 특정 움직임을 취할 때
근력 운동(움직임)에 도움을 주는 제품을 기획하고 개발 중입니다.
올해 목표는 사업자 등록이며 더 나아가 하나의 브랜드를 만들어 내는 것이 팀의 목표입니다.

Unity를 이용한 게임 개발

평소 게임 개발에 관심이 있던 대학생들이 모여 만들어진 Unity를 이용한 게임 개발 팀입니다.
서로 각기 다른 분야의 전공 학생들이 모여 Metroidvania 장르의 게임을 기획 중입니다.
팀원 모두가 기획에 참여하고 최종 목표는 게임 플랫폼에 게임 등록입니다.

Android Studio & Kotlin를 이용한 앱 개발

S/W융합 캡스톤 디자인 과목에서 진행하는 프로젝트로
1인 개발 프로젝트로 Android Studio / Kotlin을 이용한 '대학생 졸업 이수 학점 관리 앱' 개발을 준비 중입니다.
여러 대학교의 졸업 이수 학점의 기준을 사용자가 정리할 수 있게 해주어 따로 자신의 학점 관리 계획을 위해
추가적인 외부 도움을 받지 않고 계획을 짤 수 있도록 도와주는 서비스를 기획하였으며 현재 기획 마무리 단계입니다.

웹 서비스 개발

현재 '오픈소스개발방법론' 수업에서 팀 프로젝트를 준비중입니다.
내용은 '정신질환을 앓는 현대인의 정신의학 관련 기관 안내 및 매칭 서비스' 입니다.
3명의 팀원으로 이루어진 팀으로 팀원 전체가 개발을 담당합니다.
현재 기획된 내용으로는 웹 서비스를 개발할 예정입니다.
